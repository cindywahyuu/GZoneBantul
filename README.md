# GZoneBantul
Dashboard ini merupakan media visualisasi hasil analisis spasial terkait pemetaan zona kerentanan wilayah gempabumi dan analisis rute evakuasi. Dashboard dibuat untuk membantu pengguna dalam memahami tingkat kerentanan wilayah serta alternatif jalur evakuasi menuju lokasi tujuan evakuasi secara lebih interaktif dan informatif.
